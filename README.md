# Pi_Parser
Парсер — это программа для сбора и систематизации информации, размещенной на различных сайтах. Источником данных может служить текст, HTML-код сайта, заголовки, пункты меню, базы данных и т. д. 
***
Процесс работы парсера:
1) Сканирование HTML-кода, текста, базы данных и т. д.
2) Сбор данных по заданным параметрам (заголовком, ссылок, пунктов меню и т. д.)
3) Конвертация полученных данных в формат, удобный для чтения
***
Доступ парсера к сайту возможен через протокол HTTP, HTTPS или веб-браузер
